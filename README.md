# vigil - Showcase

> Self-scheduling follow-up MCP server. No external scheduler, no gateway, no cron.

[![Live Site](https://img.shields.io/badge/site-live-blue)](https://fbratten.github.io/vigil-showcase/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## What is vigil?

vigil is an MCP server that schedules and monitors follow-up tasks autonomously. 11 tools, 5 check types, 13 notification channels. 148 tests.

## Source Repository

Development happens at [fbratten/vigil](https://github.com/fbratten/vigil).

This showcase repository is the documentation and demo site. It is not open for collaborative development.
